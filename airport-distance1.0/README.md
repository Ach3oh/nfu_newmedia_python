# 简介 
输入方面用户可通过输入想查询里程的全球的两个任意机场的英文名，输出方面则是这两个任意机场间的里程。数据来源于？？？json档。
操练Python语言开发练习：使用flask




## 输入：
用户在机场1，机场2，分别输入两个不同的机场英文名。
## 输出：
通过输入两个不同的机场名，用户得到输出结果为：两个机场
## 从输入到输出，除了flask模块，本组作品使用了：
